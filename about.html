What I've been Learning?

  • HTML, CSS, and JavaScript:- For the user interface and interactivity, understanding HTML, CSS, and JavaScript is essential. Resources like Mozilla Developer Network (MDN) provide comprehensive guides and documentation. [MDN Web Docs]
  • Programming Logic and Problem Solving:- Books like "Clean Code: A Handbook of Agile Software Craftsmanship" by Robert C. Martin can help improve coding practices and problem-solving skills. [Clean Code]
  • Database Management:- To manage user accounts and transaction data, knowledge of database systems like MySQL or PostgreSQL is valuable. Websites like W3Schools provide tutorials on SQL. [W3Schools SQL Tutorial]
  • System Design and Flowcharting:- "Head First Design Patterns" by Eric Freeman and Elisabeth Robson is a great resource for understanding design principles. [Head First Design Patterns]
  • Security Practices:- Learn about secure coding practices to protect user data. "Web Application Security: A Beginner's Guide" by Bryan Sullivan is a good starting point. [Web Application Security: A Beginner's Guide]
  • Project Management:- For managing your project effectively, tools like Jira or Trello can be useful. "Scrum: The Art of Doing Twice the Work in Half the Time" by Jeff Sutherland is a book on agile project management. [Scrum: The Art of Doing Twice the Work in Half the Time]
  • Version Control:- Utilizing version control systems like Git is crucial. GitHub provides extensive documentation. [GitHub Docs]
  • Testing and Debugging:- Learn about testing frameworks and debugging tools. "The Art of Software Testing" by Glenford J. Myers is a classic in software testing. [The Art of Software Testing]
